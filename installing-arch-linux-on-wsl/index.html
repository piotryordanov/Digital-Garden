Install Arch linux under WSL is quite straightforward.

  1. Head over to ArchWsl and download the latest version
  2. Extract the file
  3. Launch a terminal (I highly recommend windows terminal)
  4. cd into the location of the arch.exe file
  5. Run it!

And just like that, arch is installed in wsl. There are further steps needed to turn it into a full fledged OS, but the above is way easier than the traditional installation steps.

Import / Export

One of the coolest things about WSL is the ability to export then import the distro. This is how I move it from one PC to another when a lot of changes happened.
That way, I end up finding myself on the new PC with the exact same files and environment as the one on the old PC!

Export

First, export the distro:

1

wsl --export <WSL Image Name> <Export file>

You can find the <WSL Image Name> by running wsl --list
